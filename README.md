# 프로젝트 소개
- 한 줄 소개: NestJS를 이용하여 Trello와 같은 칸반 보드 서비스를 위한 API 서버를 제작.
- 배경: NestJS는 규모가 크고 복잡한 프로젝트에 적합한 프레임워크. 이를 직접 느껴보기 위해 칸반 보드 서비스를 위한 API서버를 제작.
- 프로젝트 인원: 5명
- 진행 기간: 7일
